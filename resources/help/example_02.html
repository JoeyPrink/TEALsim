Example_02:

We create a simple simulation of an object falling under gravity and bouncing off of a floor.

For the object we use an EMObject, the teal.physics.em.PointCharge object. The point charge is given a positive charge (which is why it is orange), but since there are no other electromagnetic objects to interact with, the dynamics of the object are simply determined by gravity.

The collision with the floor is perfectly elastic, and there is no friction in the model, so the motion repeats indefinitely.
